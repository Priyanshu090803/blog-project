BlogApp 2.0 is a modern, full-stack blog built with a focus on speed, clarity, and a pleasant reading experience.  
The goal was to create something that feels simple, calm, and inviting, while still being fully featured on the admin side.

## Tech Stack

- Framework: Next.js 15 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- Database: PostgreSQL
- ORM: Prisma
- API Layer: tRPC (end-to-end type safety)
- Animations: Motion (Framer Motion)
- Icons: Lucide React
- Deployment: Vercel (recommended)
